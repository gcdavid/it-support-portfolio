## Projects

### 1. File Share Permissions Lab
**Location:** `projects/file-share-permissions-lab`

This project demonstrates the creation and securing of departmental file shares (HR, IT, and Sales) using proper NTFS and Share permissions in a Windows Active Directory environment. It includes troubleshooting access denied issues and applying best practices for permission management.

**Key Skills:** NTFS/Share Permissions, Active Directory Security Groups, Access Troubleshooting, Documentation

### 2. Group Policy - Corporate Workstation Management Lab
**Location:** `projects/group-policy-workstation-lab`

This project focuses on centrally managing and securing Windows workstations using Group Policy. It covers security hardening (password policies, account lockout, USB blocking, Control Panel restrictions), user experience standardization (corporate wallpaper, mapped drives), and software deployment. Multiple real-world troubleshooting scenarios were encountered and resolved during implementation.

**Key Skills:** Group Policy Management, Security Hardening, GPO Troubleshooting, Drive Mapping, Software Deployment
